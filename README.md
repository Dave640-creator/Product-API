# Product API

## Table Structure
- id
- product
- price

## Endpoints

| Endpoint | Method | Action |
|----------|--------|--------|
| /api/products | GET | Get all products |
| /api/products?id=1 | GET | Get single product |
| /api/products | POST | Create product |
| /api/products | PUT | Update product |
| /api/products | DELETE | Delete product |

## Description
Simple REST API for products (no database, simulation only).
